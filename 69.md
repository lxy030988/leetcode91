# 思路

- 二分查找

# 代码 js

```js
/**
 * @param {number} x
 * @return {number}
 */
var mySqrt = function (x) {
  if (x == 1) return 1
  let left = 1
  let right = x >> 1
  while (left <= right) {
    const middle = (left + right) >> 1
    const res = middle * middle
    if (res == x) {
      return middle
    } else if (res > x) {
      right = middle - 1
    } else if (res < x) {
      left = middle + 1
    }
  }
  return (left + right) >> 1
}
```

# 复杂度分析

- 时间复杂度：O(logn)
- 空间复杂度：O(1)
