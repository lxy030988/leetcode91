# 思路

- 暴力解法,两层 for 循环
- 把字符串暂存,判断后续字符有没有重复

# 代码 js

```js
/**
 * @param {string} s
 * @return {number}
 */
var lengthOfLongestSubstring = function (s) {
  if (s.length == 0) return 0
  if (s.length == 1) return 1
  let res = 0,
    temp,
    tempstr
  for (let i = 0; i < s.length - 1; i++) {
    temp = 1
    tempstr = s[i]
    for (let j = i + 1; j < s.length; j++) {
      if (tempstr.includes(s[j])) {
        break
      }
      tempstr += s[j]
      temp++
    }
    if (temp > res) {
      res = temp
    }
  }
  return res
}
```

# 复杂度分析

- 时间复杂度：O(n3)
- 空间复杂度：O(n)
