# 思路

- 遍历每一步的操作

# 代码 js

```js
/**
 * @param {string} moves
 * @return {boolean}
 */
var judgeCircle = function (moves) {
  let x = 0,
    y = 0
  for (const move of moves) {
    switch (move) {
      case 'L':
        x--
        break
      case 'R':
        x++
        break
      case 'D':
        y--
        break
      default:
        y++
        break
    }
  }
  return x === 0 && y === 0
}
```

# 复杂度分析

- 时间复杂度：O(n)
- 空间复杂度：O(1)
