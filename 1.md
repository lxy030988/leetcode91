# 思路

- 遍历数组,把每个数据暂存到 map 中
- 从 map 中取 (target - 当前值)

# 代码 js

```js
/**
 * @param {number[]} nums
 * @param {number} target
 * @return {number[]}
 */
var twoSum = function (nums, target) {
  const map = new Map()
  for (let i = 0; i < nums.length; i++) {
    const num = nums[i]
    if (map.has(target - num)) {
      return [map.get(target - num), i]
    } else {
      map.set(num, i)
    }
  }
}
```

# 复杂度分析

- 时间复杂度：O(n)
- 空间复杂度：O(n)
